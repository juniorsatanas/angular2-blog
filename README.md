# Angular2-blog
This project uses ES6.

## Requirements

```
npm install -g http-server
npm install
```

## ES6
Located in `app/`

```
http-server ./app
```

go to `http://localhost:8080`

## Details
It uses [http://gokhankaradas.com/wp-json/posts](http://gokhankaradas.com/wp-json/posts) backend api.
